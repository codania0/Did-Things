Did Things – Politique de confidentialité
Dernière mise à jour : 13 février 2026
Did Things accorde une grande importance à la protection de vos données personnelles. La présente politique explique quelles données sont collectées, pourquoi elles le sont et comment elles sont utilisées.
1. Données collectées
Nous collectons uniquement les données strictement nécessaires au fonctionnement de l’application :
Adresse e-mail
Identifiant utilisateur (UID Firebase)
Contenu saisi par l’utilisateur dans l’application (actions enregistrées)
Données techniques minimales liées à l’authentification
Aucune donnée n’est collectée à des fins publicitaires.
L’application n’utilise pas de services d’analyse (Analytics) ni de suivi comportemental.
2. Finalité du traitement
Les données sont utilisées exclusivement pour :
Créer et gérer le compte utilisateur
Permettre la connexion (email/mot de passe, Apple, Google)
Sauvegarder et afficher le contenu saisi
Assurer la sécurité du compte
Le traitement des données est nécessaire à la fourniture du service demandé par l’utilisateur.
3. Services tiers
L’application utilise les services suivants :
Firebase Authentication (Google LLC)
Sign in with Apple (Apple Inc.), si choisi par l’utilisateur
Google Sign-In (Google LLC), si choisi par l’utilisateur
À ce jour, l’application n’utilise pas Firebase Analytics, Crashlytics ni Google Analytics.
Ces services peuvent traiter certaines données d’authentification conformément à leurs propres politiques de confidentialité.
4. Stockage et sécurité
Les données sont hébergées via Firebase (Google LLC).
Nous mettons en œuvre des mesures raisonnables de sécurité afin de protéger les données contre tout accès non autorisé, perte ou divulgation.
5. Conservation des données
Les données sont conservées tant que le compte utilisateur est actif.
Elles sont supprimées définitivement lors de la suppression du compte.
6. Suppression du compte
L’utilisateur peut supprimer son compte directement depuis l’application :
Profil > Supprimer mon compte
La suppression est définitive et entraîne la suppression de l’ensemble des données associées au compte, y compris les données d’authentification.
7. Vos droits (RGPD)
Conformément au Règlement Général sur la Protection des Données (RGPD), vous pouvez :
Demander l’accès à vos données
Demander la rectification de vos données
Demander la suppression de vos données
8. Contact
Pour toute question relative aux données personnelles :
4nia.pro@gmail.com

Did Things – Privacy Policy
Last updated: February 13, 2026
Did Things values your privacy. This policy explains what data is collected, why it is collected, and how it is used.
1. Data We Collect
We only collect data strictly necessary to operate the application:
Email address
User ID (Firebase UID)
User-entered content within the app (logged actions)
Minimal technical authentication data
No advertising data is collected.
The app does not use analytics services or behavioral tracking.
2. Purpose of Processing
Data is used exclusively to:
Create and manage user accounts
Enable sign-in (email/password, Apple, Google)
Save and display user-entered content
Ensure account security
Data processing is necessary to provide the service requested by the user.
3. Third-Party Services
The application uses:
Firebase Authentication (Google LLC)
Sign in with Apple (Apple Inc.), if selected by the user
Google Sign-In (Google LLC), if selected by the user
The app does not use Firebase Analytics, Crashlytics, or Google Analytics.
These providers may process authentication-related data in accordance with their own privacy policies.
4. Storage and Security
Data is hosted via Firebase (Google LLC).
Reasonable safeguards are implemented to protect data against unauthorized access, loss, or disclosure.
5. Data Retention
Data is retained as long as the user account remains active.
It is permanently deleted upon account deletion.
6. Account Deletion
Users can delete their account directly within the app:
Profile > Delete my account
Deletion is permanent and removes all account-related data, including authentication credentials.
7. Your Rights (GDPR)
Under the General Data Protection Regulation (GDPR), you may:
Request access to your data
Request correction of your data
Request deletion of your data
8. Contact
For privacy-related inquiries:
4nia.pro@gmail.com
